MVP Plataforma de consultoría política · Semana 1

Objetivo de la semana 1

Establecer la base técnica del producto: repositorio, entornos, arquitectura desacoplada (Django + DRF + React + MUI + PostgreSQL) y la estructura general lista para iterar en las semanas siguientes.

Enfoque

  • Configuración inicial del proyecto y control de versiones.
  • Entorno de desarrollo reproducible (local y/o Docker).
  • Arquitectura backend / frontend y convenciones de API (/api/v1).
  • Base de datos PostgreSQL y modelo de datos inicial alineado al MVP.
  • Estructura general del sistema (apps, permisos por rol, healthcheck).

Entregables esperados

  • Proyecto inicial configurado y documentado en README.
  • Arquitectura técnica base acordada (stack, despliegue objetivo, variables de entorno).
  • Entorno de desarrollo operativo (servidor API + cliente + DB accesibles).
